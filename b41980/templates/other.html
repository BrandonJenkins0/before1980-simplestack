House Built Before 1980 API Service

Simple API Service that let's you send new instances of houses against my random forest model built in scikit-learn.

Endpoints

  • / or /about - shows info about the api service
  • /isAlive - shows "Yessir!" if service is up and running
  • /predict - api where you send params of a new house instance and it returns a prediction

    • Parameters Needed

  • livearea => int
  • stories => int
  • bdrms => int
  • baths => int

    • Example URL Request

  • http://localhost:1001/predict?livearea=2401&stories=2&bdrms=3&baths=1

    • Returns

  • An array of features provided by you in the api call.
  • A 1 if house is built before 1980 and 0 otherwise.
